Developed with Unreal Engine 5.

This is an example project for the [IPFS Plugin](https://www.unrealengine.com/marketplace/en-US/product/blockchain).

This example project requires additional plugins to be installed into your engine, all of these required plugins are
forever free on the unreal marketplace.

All future examples and implementations will be introduced into this project.

:::info
The source files of the `IPFSExample` unreal project can be downloaded
[here](https://dev.azure.com/3StudioOnline/Public%20Resources/_git/Web3UE).
:::
